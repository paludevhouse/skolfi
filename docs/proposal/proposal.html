1. Ringkasan Eksekutif

SkolFi adalah sistem keuangan sekolah siap-pakai yang mengelola seluruh siklus penagihan — dari generate tagihan SPP per kelas, pembayaran tunai & online, tunggakan, potongan, beasiswa, hingga laporan pendapatan dan kartu pembayaran. Terdiri dari dua portal: panel Admin untuk staf sekolah & kasir, serta portal Siswa/Orang Tua untuk melihat tagihan dan melakukan pembayaran online.

Yang termasuk dalam penawaran ini

  • Pengembangan sistem lengkap — seluruh modul yang dirincikan di bagian 2, siap digunakan.
  • Hosting Railway 2 bulan pertama — deployment production, database PostgreSQL, dan domain.
  • Konfigurasi awal & data master — tahun ajaran, kelas, layanan biaya, akun admin.
  • Handover & pelatihan singkat — sesi demo penggunaan untuk admin & kasir.
Teknologi: Next.js 14, React 18, PostgreSQL (Prisma ORM), Mantine UI, TanStack Query, JWT auth, deployment Railway. Mendukung multi jenjang sekolah (TK / SD / SMP / SMA) dan dwibahasa (Indonesia / English).

2. Rincian Modul & Menu

Di bawah ini adalah daftar modul yang tersedia dalam sistem, tersusun sesuai struktur menu aplikasi.

Utama

Dashboard · Pembayaran (tunai) · Tagihan Layanan · Siswa
Ringkasan Quick Actions

Akademik

Tahun Ajaran · Kelas · Penugasan Siswa ke Kelas
Multi tahun ajaran TK / SD / SMP / SMA

Keuangan

Tagihan SPP (Tuitions) · Potongan (Discounts) · Beasiswa · Biaya Layanan · Layanan Biaya (Transport, Akomodasi)
Bulk generate Import Excel Per-kelas / per-siswa

Laporan

Laporan Tunggakan · Ringkasan Kelas · Ringkasan Layanan · Laporan Transport · Laporan Pendapatan (harian/bulanan/tahunan) · Kartu Pembayaran
Export Excel Filter NIS + jenjang

Sistem

Karyawan (Admin / Kasir) · Manajemen Akun · Bantuan
RBAC JWT Auth

Portal Siswa / Orang Tua

Beranda tagihan · Riwayat transaksi · Ganti password
Login via NIS Mobile-friendly

Fitur Lintas Modul

  • Bulk Import Excel — siswa, potongan, beasiswa, biaya layanan, generate tagihan.
  • Export Excel — seluruh laporan keuangan, format rapi, siap cetak/arsip.
  • Pencarian & Filter — NIS, nama, jenjang sekolah (TK/SD/SMP/SMA), tahun ajaran, kelas.
  • Role-Based Access — peran Admin (penuh) dan Kasir (terbatas pada pembayaran & laporan).
  • Keamanan — JWT 8 jam, blacklist token saat logout, rate limiting per endpoint, idempotency.
  • Dwibahasa — Indonesia & English via next-intl.
